Let's get you set up. This workspace allows you to generate models and print them directly to a Bambu Lab printer.

Printer Configuration

To enable direct slicing and printing (bypassing Bambu Studio), configure your local printer settings.

1. In your project root, copy the template:
cp .env.example .env

2. Edit the .env file and enter your printer's details:
- BAMBU_PRINTER_IP (e.g. 192.168.1.157)
- BAMBU_PRINTER_SERIAL (from printer info)
- BAMBU_PRINTER_ACCESS_CODE (8-digit LAN code)

3. Restart with docker compose up --build
The status indicator in the top-right header will turn Green (Online) when the connection is established. You can click it any time to view this guide again.
