piSlice - stackable HAT case for FLIRC Pi Zero metal case (includes case for Waveshare Ethernet / USB HUB HAT) by TMfantomas on Thingiverse: https://www.thingiverse.com/thing:4752623

Summary:
Stackable case for extra HATs to be added on top of FLIRC's Raspberry Pi Zero case.I've included three case models plus USB-brigde-cover (for Waveshare's Ethernet / USB HUB HAT):TEMPLATE - this is a basic piSlice case template to be further modified (cutting or stretching middle layers to change overall height to fit particular HAT and to add needed cutouts),DESKTOP - variant of piSlice is designed for Waveshare's Ethernet / USB HUB HAT to be used as a desktop case,DIN - variant of DESKTOP piSlice case but with additional holes on the back wall spaced at 25 &amp; 37 mm to accommodate some of the cheap plastic DIN rail clips (holes should fit M4 bolts)Not sure what the tolerances are for Flirc's cases since mine didn't exactly match provided dimensions.
